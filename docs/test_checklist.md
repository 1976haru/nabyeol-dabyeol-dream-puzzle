# 사람이 직접 확인할 테스트 체크리스트

배포 전/후에 아래 항목을 직접 확인하세요.

## 게임 루프
- [ ] 블록을 선택하면 강조 표시가 된다
- [ ] 인접한 블록과 교환되어 3개 이상 매치가 사라진다
- [ ] 매치 후 위 블록이 아래로 떨어진다 (낙하)
- [ ] 빈 칸이 새 블록으로 채워진다
- [ ] 연쇄(cascade)가 일어나면 추가 점수가 들어온다
- [ ] 매치가 안 되는 교환은 무효 처리된다 (선택만 바뀜)
- [ ] 힌트 버튼을 누르면 가능한 수가 노란색으로 표시된다

## 스테이지 / 판정
- [ ] 스테이지 1~10 모두 진입 가능
- [ ] 목표 점수 도달 시 클리어 팝업
- [ ] 이동 횟수 소진 + 목표 미달 시 실패 팝업
- [ ] 클리어 후 "다음 스테이지" 이동
- [ ] 잠긴 스테이지는 🔒 로 비활성, 이전 스테이지 클리어 시 해금
- [ ] 도감에서 스테이지 3/5/7/10 클리어 시 캐릭터 해금 확인

## 저장 / 진행도
- [ ] 새로고침 후에도 진행도(클리어/최고점/도감) 유지
- [ ] "이어하기" 가 마지막 스테이지로 이동
- [ ] 소리 on/off 설정이 유지된다

## 반응형 / 모바일
- [ ] 360px / 390px / 430px 폭에서 보드가 깨지지 않는다
- [ ] Android Chrome 에서 터치로 정상 조작
- [ ] 모든 글자가 16px 이상으로 읽기 편하다

## PWA
- [ ] 홈화면에 추가(설치) 가능
- [ ] 설치 후 전체화면(standalone)으로 실행
- [ ] 오프라인에서도 한 번 로드한 게임이 실행됨
